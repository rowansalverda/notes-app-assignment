### Notes App Assignment

This is a simple notes app built with HTML, CSS and Node.js for my Software Development Bootcamp with Circuit Stream and the University of British Columbia. 

### File Structure

Notes App Assignment/
│
├── frontend/

├── models/

├── routes/
│   ├── authRouter.js 
│   └── notesRouter.js
├── .gitignore
├── server.js
├── package.json
└── package-lock.json
```

### How to run the app

1. Clone the repository

2. Install the dependencies: 
npm install 

3. Run the application
node app.js

4. Open your web browser ad go to: 
http://localhost:3000
